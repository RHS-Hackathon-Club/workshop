9 UGC

9.1 Handling When Saving UGC to Shared Save Data

When applications save UGC (either created by the user or received from elsewhere) to shared save data, depending on the application's specifications this UGC may unintentionally be viewed or distributed to users on other accounts registered on the same system.

As a result, if your application saves UGC to shared save data, you must must fully investigate to ensure that issues will not arise even if users on other accounts can access this UGC.

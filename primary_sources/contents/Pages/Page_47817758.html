7.1 Image Data Attached to Posts to Communities or Messages to Friends

Image data attached to posts to communities or messages to friends must comply with the following format requirements.

Item Format
Image format

Baseline JPEG format or MPO format
Size

Baseline JPEG: 320 × 240, 400 × 240, 640 × 480, or 800 × 450 (all dimensions are width × height)

MPO: 400 width × 240 height

7.2 Handwritten Messages Attached to Posts to Communities or Messages to Friends

Handwritten messages attached to posts to communities or messages to friends must comply with the following format requirements.

Item Format
Image format 32-bit TGA file with alpha channel (R,G,B,A)
Size 320 width × 120 height
RGBA

Pixel color must be either 0xFFFFFFFF (white) or 0x000000FF (black).

7.3 Community Icons

Community icons must comply with the following format requirements.

Item Format
Image format 32-bit TGA file with alpha channel (R,G,B,A)
Size 128 width × 128 height
RGBA Alpha value of 0xFF
